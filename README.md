# Digit_Recognition
Use of PCA, SVM, to train program with Digits and make predictions, Currently 95% accuracy. Python 3

### GENERAL NOTES:
    Training Data comes from Scit-Learn package import
    
## DEPENDENCIES:
   1. Numpy
   1. SciKit-Image AND SKlearn 
   1. Matplotlib
   1. Scipy
   1. PYTHON 3
